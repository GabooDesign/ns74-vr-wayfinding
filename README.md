# MetroSantiagoSimulador — NS-74 Investigation
**_(En desarrollo)_**

Plataforma WebXR orientada al análisis interactivo de señalética, percepción espacial y wayfinding en entornos ferroviarios del Metro de Santiago (NS-74).

Proyecto de investigación enfocado en reconstrucción virtual interpretativa, visualización inmersiva y análisis de orientación espacial en interiores ferroviarios.
